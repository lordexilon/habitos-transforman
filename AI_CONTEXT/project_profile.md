# Perfil del Proyecto

**Nombre**: Hábitos Poderosos (habitos-transforman)
**Stack Tecnológico**:
- **Framework**: Next.js (App Router), React
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS
- **PWA**: @ducanh2912/next-pwa
- **Iconografía**: lucide-react
- **Base de Datos / Autenticación**: Supabase (@supabase/supabase-js, @supabase/ssr)

**Arquitectura**:
- Monolito Serverless (Next.js App Router).
- Desarrollo "Mobile First" enfocado en PWA.

**Estrategia de Búsqueda Recomendada**:
- Buscar lógica de rutas y vistas en `src/app/`.
- Buscar componentes reusables en `src/components/`.
- Buscar configuración de base de datos en `src/lib/`.
- Buscar contenido dinámico en la carpeta `/content/`.
